⚗️ Perfume Store Setup

Hubungkan ke Google Sheets sebagai database agar data tersimpan & bisa diakses dari perangkat mana pun.

Step 1 — Buat Google Sheet

Buka sheets.google.com → buat spreadsheet baru → beri nama bebas (mis: Perfume Store DB)

Step 2 — Deploy Apps Script

Di Google Sheet: Extensions → Apps Script → hapus semua kode → paste kode dari tutorial → klik Deploy → New Deployment → pilih Web App → set akses Anyone → copy URL-nya

Step 3 — Paste URL di bawah

Klik tab Tutorial setelah masuk untuk panduan lengkap.

🌐 Cara Host GRATIS di GitHub Pages

1. Daftar akun di github.com
2. Klik New Repository → beri nama (mis: perfume-store) → centang Public
3. Upload file perfume-store.html → rename jadi index.html
4. Buka Settings → Pages → Source: main branch → Save
5. URL mu jadi: https://username.github.io/perfume-store — GRATIS & permanen!
✓ Bisa diakses dari HP, laptop, tablet mana pun!

📊 Setup Google Sheets sebagai Database

2. Buat 3 sheet dengan nama persis: Produk, Penjualan, Pembelian
3. Di menu Extensions → Apps Script
